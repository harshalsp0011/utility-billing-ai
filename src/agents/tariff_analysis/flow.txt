Tariff Extraction Flow (repo-root relative paths)

1) Extract PDF text (page-wise)
	- Script: src/agents/tariff_analysis/extract_pdf.py
	- Input: data/raw/<your_tariff.pdf> (or pass a path via CLI)
	- Output: data/processed/raw_extracted_tarif.json
	- Run: venv/bin/python src/agents/tariff_analysis/extract_pdf.py [optional /absolute/path/to/pdf]

2) Classify pages and extract sections per Service Class
	- Script: src/agents/tariff_analysis/sc_section_extractor.py
	- Input: data/processed/raw_extracted_tarif.json
	- Outputs:
		 • data/processed/harshal_scheme_pages.json
		 • data/processed/harshal_sc.json
	- Run: venv/bin/python src/agents/tariff_analysis/sc_section_extractor.py

3) Generate final tariff JSON via LLM
	- Script: src/agents/tariff_analysis/Harshal_sc_llm_extractor.py
	- Input: data/processed/harshal_sc.json
	- Outputs:
		 • data/processed/Hp_sc_final_tariffs.json
		 • data/processed/Hp_sc_llm_raw_responses.json (for debugging/auditing)
	- Run: venv/bin/python src/agents/tariff_analysis/Harshal_sc_llm_extractor.py
	- Notes: Requires valid OPENAI_API_KEY (and OPENAI_MODEL) in environment/.env

4) (Optional) Ingest final tariffs into the database
	- Script: tests/ingest_tariff_json.py
	- Input (default): data/processed/Hp_sc_final_tariffs.json
	- Run: venv/bin/python tests/ingest_tariff_json.py
